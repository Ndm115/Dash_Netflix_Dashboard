# Dash Netflix Dashboard

An interactive dashboard for analyzing Netflix content using **Dash** and **Plotly**. Visualize trends, genres, content types, durations, and global distribution of movies and TV shows.

---

## Features

- Filter by release year and content type (Movie/TV Show)
- Top 10 genres chart
- Timeline of content added over time
- Pie chart of Movies vs TV Shows
- Duration trends scatter plot
- Choropleth map showing content by country

---
